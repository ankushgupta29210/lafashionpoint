Firebase Setup

To connect this admin panel to Firebase:

  1. Go to console.firebase.google.com and create a project
  2. Enable Firestore Database (Build → Firestore → Create)
  3. Enable Authentication → Email/Password
  4. Enable Storage
  5. Go to Project Settings → Your apps → copy the config
  6. Paste values into js/firebase-config.js
  7. In Firebase Auth, create an account for yourself (you@email.com + password)
  8. Add Firestore and Storage rules from FIREBASE_RULES.md

Firestore Security Rules

Paste these rules in Firebase Console → Firestore → Rules:

rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    match /products/{id} {
      allow read: if true;
      allow write: if request.auth != null;
    }
    match /inquiries/{id} {
      allow create: if true;
      allow read, update, delete: if request.auth != null;
    }
    match /orders/{id} {
      allow read, write: if request.auth != null;
    }
    match /coupons/{id} {
      allow read: if true;
      allow write: if request.auth != null;
    }
  }
}

Storage Security Rules

Paste these rules in Firebase Console → Storage → Rules:

rules_version = '2';
service firebase.storage {
  match /b/{bucket}/o {
    match /products/{imageId} {
      allow read: if true;
      allow write: if request.auth != null;
    }
  }
}
